# Market-Sentiment-Impact-on-Trader-Performance-Project
This project analyzes how market sentiment (Fear, Greed, Extreme Greed, Neutral) affects trader performance and behavior using real trading data.

## What This Project Covers
- Data cleaning and merging sentiment with trading data
- Exploratory Data Analysis (EDA)
- Performance comparison across market moods
- Trading behavior analysis
- Trader segmentation
- Strategy recommendations

## How to Run
1. Open the notebook in Jupyter or Google Colab
2. Run cells from top to bottom

## Key Outcome
Market sentiment strongly influences trading activity, profit levels, and risk behavior.  
Fear periods showed higher activity and profit opportunities, while Greed and Extreme Greed required more careful strategies.
